# 病原体厌恶

## 适用性门槛

必须满足：
- 可见的有机污染、腐烂、疾病、废弃物、体液、伤口、变质食物、寄生虫或虫害在情感上居于中心位置。
- 回避／排斥比攻击威胁、悲伤或构图更能解释该反应。
- 线索身份由视觉证据支持，而不是根据标签或背景故事进行推断。

排除：
- 缺乏可见病原体线索的道德或社会违规。
- 蛇、捕食者或攻击形态占据主导。
- 新鲜且具有食欲吸引力的食物、洁净医疗设备，或不涉及污染的无害昆虫。

近似但不适用：
- 捕食或攻击 -> evolved-fear-module。
- 当与人的利害关系占主导时的受伤场景 -> cognitive-appraisal；只有在污染居于中心时才保留在此技能。
- 缺乏有机污染的混乱抽象图像 -> berlyne-arousal-pleasure 或 none。

## 视觉变量

- **线索类型**：腐败、霉菌、分解、脓液、呕吐物、粪便、积聚的血液、暴露组织、变质食物、寄生虫、虫害。
- **范围／密度**：局部痕迹与大范围污染或密集虫害之间的区别。
- **接触可能性**：触摸、摄入相关性、接近程度、渗漏、是否被容器隔离。
- **材料状态**：湿润、黏液、渗出、变色、纹理破坏。
- **情境削弱因素**：医疗护理、标本、发酵、包装、虚构内容、屏障。

## 推断流程

1. 识别可见的病原体线索；将观察事实与不确定识别结果分开。
2. 根据可见证据估计范围、接触可能性和降解程度。
3. 将污染与攻击威胁以及经过处理的新鲜伤口区分开。
4. 应用医疗／烹饪／科学／虚构情境的削弱因素，但不要因此抹去生动线索。
5. 使用锚定范围判断 VA。厌恶会稳定地降低效价；它对唤醒度的提升要小得多。

## 唤醒度门槛

厌恶是一种回避反应，而不是警报反应：大多数污染内容的唤醒度位于 4-5.2。
只有在接触迫近、新鲜血腥内容／暴露组织，或污染失控地向观察者扩散时，
才将唤醒度提高到 5.5 以上。静态腐烂、脏乱或污渍都不是激活线索。
不存在激活线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 严重身体血腥内容：肢体损毁、组织暴露、新鲜伤口 | 1.5-2.5 | 5.5-6.8 |
| 强烈腐烂／废弃物／虫害，但没有身体组织暴露 | 2.5-3.5 | 4.0-5.2 |
| 轻度或被隔离的污染 | 3.5-4.5 | 3.4-4.4 |
| 强削弱因素（医疗、标本、虚构） | 3.8-5.0 | 3.2-4.2 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，应停留在该
区间中最接近中性值的一端。

## 示例

- **腐烂型**：面包表面覆盖绿色—黑色霉菌和潮湿变色区域，并且放在一个敞开的袋子中。->
  变质食物、大范围生长、与摄入相关、没有身体组织暴露；
  activation_evidence []。**V 2.8，A 4.4**，证据强。
- **血腥型**：一个深而开放的伤口特写，可以看到暴露组织和新鲜血液。-> 严重的
  身体线索、高度生动且具有接触显著性；activation_evidence ["新鲜暴露的
  组织"]。**V 2.0，A 6.2**，证据强。

## 输出约定

在共享 JSON 中分别报告线索类型、范围／接触、真实性、削弱因素，以及顶层
`activation_evidence`（如果不存在则为空列表）。描述材料本身，绝不要给某个人贴上“令人恶心”的标签；
应说明识别的不确定性。
