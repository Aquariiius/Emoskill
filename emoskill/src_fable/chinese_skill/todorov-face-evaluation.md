# Todorov 面孔评估

## 适用性门槛

必须满足：
- 清晰的人类面孔是占主导地位的主体，并且周围事件情境很少。
- 表情处于中性、静止或足够轻微的状态，使结构性第一印象仍具有相关性。
- 用户明确请求社会印象特征，而不是观察者 VA 或情绪识别。

排除：
- 强烈微笑、哭泣、愤怒、恐惧、触碰、冲突、亲密或情境性动作驱动解释。
- 身体姿势、婴儿图式、遮挡、风格化处理或周围情境占据主导。
- 任务要求判断真实品格、意图、可信度、危险性、能力、诊断结果或观察者 VA。

近似但不适用：
- 依赖情境的人脸场景 -> cognitive-appraisal。
- 全身动作信号 -> emotional-body-language。
- 婴儿化形态 -> baby-schema。

## 视觉变量

- **面孔主导程度／可见性**：在画面中的大小、遮挡程度、分辨率、角度和光照。
- **表情强度**：中性／静止状态与清晰情感表情之间的区别。
- **温暖感／威胁相似性**：可见的眉毛、眼部区域紧张程度和静止嘴部构型；描述时不得作出人格断言。
- **支配性相似性**：面部成熟度、宽度、下颌突出程度和姿态；应将其视为感知到的结构，而不是能力。
- **情境污染**：服装、场景、互动、身份线索和刻板印象，这些因素可能会使印象产生偏差。
- **可靠性限制**：图像质量、风格化处理、人口统计偏差，以及外貌与真实人物之间的不匹配。

## 推断流程

1. 确认用户明确要求使用该模块，并确认面孔占主导、可见性良好、情境较少且表情为中性／轻微。
2. 在使用社会评价术语之前，先记录结构性观察结果。
3. 估计感知可信度／效价相似性，并引用可见依据。
4. 单独估计感知支配性相似性；不要将任何一个维度转换为人格或意图。
5. 报告不确定性和偏差限制。在不输出最终观察者 VA 的情况下停止。

## 特征判断

以低、中或高等级返回感知可信度／效价相似性和支配性相似性，并同时给出置信度。这些是基于外貌形成的第一印象特征，而不是人物本身的属性。强烈表情或事件情境会使结构性解释失效，而不是成为修正因素。绝不要将这些特征转换为最终效价—唤醒度分数。

## 示例

- **图像**：一张光照良好、正面朝向的成人头部照片，嘴巴闭合并保持静止，眉毛放松，背景简单。
- **分析**：面孔可见性强，情境干扰较少；表情轻微。结构线索可能支持一个暂时性的温暖印象和中等支配性印象，但身份和品格仍然不可知。
- **输出**：感知可信度／效价相似性 = 中高；感知支配性相似性 = 中；不确定性 = 中，因为基于外貌的判断容易受到偏差影响。不返回观察者 VA。

## 输出约定

返回 `applicability`、`visual_evidence`、`perceived_trustworthiness`、`perceived_dominance`、`uncertainty` 和 `ethical_limit`。不要返回最终 `valence_score` 或 `arousal_score`。
