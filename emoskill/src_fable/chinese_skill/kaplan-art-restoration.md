# Kaplan 注意恢复理论

## 适用性门槛

必须满足：
- 一个可进入或可居住的环境是主要情感来源。
- 可见的空间组织支持低认知负荷的观看、轻松探索、休息或恢复。
- 至少有三个恢复性变量可以观察到，其中包括一个空间变量。

排除：
- 孤立物体、产品、纹理、桌面摆设或装饰性静物。
- 压倒性的尺度、猛烈力量、密集新奇性、威胁或以人物为中心的动作占据主导。
- 平静感主要来自空白或内容缺乏，而不是环境结构。

近似但不适用：
- 巨大尺度与自我渺小感 -> awe。
- 新奇性、模糊性、密集图案或硬性吸引 -> berlyne-arousal-pleasure。
- 与人有关的利害关系或可见危险 -> cognitive-appraisal。

## 视觉变量

- **表观开放性／纵深**：天空、地平线、层叠距离、路径、透视；绝不要声称实际物理米数。
- **可进入性／可供性**：路径、座椅、庇护空间、可行走表面、可使用的室内空间。
- **连贯性／延展性**：相互连接、易于理解的布局，并且具有足够丰富的视觉世界，可供心理探索。
- **柔性吸引**：水体、植被、云朵、柔和光线、重复的自然细节；这些元素能够吸引注意力，但不需要主动搜索。
- **兼容性／庇护性**：是否适合休息、漫步、观察；远景—庇护平衡。
- **注意需求**：杂乱、突兀对比、拥挤、警示线索、威胁。

## 推断流程

1. 确认承载情感的是环境，而不是物体或人物；评估可进入性。
2. 根据明确命名的单目线索估计开放性、纵深、连贯性和可供性。
3. 将柔性吸引与 Berlyne 式硬性吸引区分开；记录杂乱或奇观性。
4. 将证据整合为恢复潜力；戏剧性、暴露感或拥挤会削弱恢复潜力。
5. 使用下方锚定范围判断 VA；恢复意味着低唤醒，而不是轻微兴奋。

## 唤醒度门槛

恢复性场景按定义属于低激活：从 3 附近开始，并保持在 2.5-4.0，除非能够看到
具体的激活线索（具有力量感的流动水体、人群、风暴光线）。风景美、
丰富细节或宽广视野都不是激活线索。不存在激活线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 强恢复性环境，安全且连贯 | 6.8-7.6 | 2.5-3.8 |
| 轻度恢复性环境，普通的平静场所 | 6.0-7.0 | 3.2-4.4 |
| 风景优美但具有一定刺激性（部分匹配） | 6.0-7.2 | 4.4-5.4 |
| 匹配较弱 | 5.0-6.0 | 3.5-4.5 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，应停留在该
区间中最接近中性值的一端。

## 示例

- **平静型**：一条湖边小径穿过树林，弯曲通向开阔水面和远处山丘，场景处于柔和的
  日光中；没有人物或障碍物。-> 纵深、可步行路径、远景＋庇护、低需求；
  activation_evidence []。**V 7.3，A 3.0**，证据强。
- **部分匹配型**：一个明亮的海岸观景点，海浪被风吹动，步道上行人较多。-> 开阔且
  具有吸引力，但风、运动和拥挤增加了注意需求；activation_evidence ["运动中的海浪",
  "拥挤的步道"]。**V 6.6，A 4.8**，部分匹配。

## 输出约定

在共享 JSON 中报告空间线索、ART 构念、恢复潜力、竞争性
刺激，以及顶层 `activation_evidence`（如果不存在则为空列表）。不要声称测得了
实际尺寸、安全性或真实恢复效果。
