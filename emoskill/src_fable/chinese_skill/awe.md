# 敬畏

## 适用性门槛

必须满足：
- 物理上的广阔尺度或力量是主要的情感机制，而不只是美感或空间开阔感。
- 至少可以看到两个相互独立的尺度线索：微小的参照人物、极端的垂直尺度、无边界的延伸、深远的大气透视，或巨大的不可控制力量。
- 该场景能够合理地因尺度而引发自我渺小感、惊叹感或崇高性的威胁感。

排除：
- 平静的海岸线、森林、花园、房间或地平线；这些场景具有恢复性，但并不令人感到压倒。
- 纪念碑、纪念性场所或象征性内容；其情感由意义而非物理尺度驱动。
- 缺乏可靠尺度参照的戏剧性色彩、新奇性或广角风格。

近似但不适用：
- 低认知负荷、可进入的风景 -> kaplan-art-restoration。
- 知觉新奇性或复杂构图 -> berlyne-arousal-pleasure。
- 灾难后果或与人相关的利害关系 -> cognitive-appraisal。

## 视觉变量

- **尺度参照物**：用于建立相对大小的人、车辆、树木或建筑物。
- **垂直上的宏大尺度／纵深**：高耸的山峰、墙体、悬崖、峡谷深度或层叠的远景。
- **力量／不可控制性**：风暴、火山喷发、雪崩或猛烈的水流。
- **暴露程度／接近程度**：悬崖边缘的视角、包围性的巨大尺度或逃离困难。
- **安全性／美感**：稳定的观察位置、和谐的光线、庇护条件，以及不存在迫在眉睫的危险。

## 推断流程

1. 确认占主导地位的机制是物理尺度，而不是象征意义或美感。
2. 列出相互独立的尺度线索；如果只有镜头营造的戏剧感而没有参照物，则应排除。
3. 根据人类与环境之间的尺度差异估计自我渺小感。
4. 根据暴露程度、力量和庇护线索，将其分类为安全型敬畏、威胁型敬畏或混合型敬畏。
5. 使用锚定范围判断 VA；安静而遥远的宏大景观只会产生中等程度的激活。

## 唤醒度门槛

仅有广阔感并不是激活线索。只有在能够看到运动中的力量、包围性的近距离尺度、
暴露处境或失去控制时，才将唤醒度提高到 5 以上；稳定、遥远而宁静的全景通常
位于 4.5-5.5 附近。不存在激活线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 安全型敬畏，宁静而遥远的广阔景观 | 6.8-7.8 | 4.4-5.6 |
| 安全型敬畏，具有包围感或戏剧性的巨大尺度 | 6.6-7.6 | 5.6-6.6 |
| 威胁型敬畏：猛烈力量、暴露处境、无法控制 | 3.0-4.5 | 6.0-7.5 |
| 尺度证据较弱 | 5.0-6.0 | 4.0-5.0 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，
应停留在该区间中最接近中性值的一端。

## 示例

- **安全型**：两名显得十分微小的徒步者站在稳定的观景台上，在清晰的光线中面对一片巨大的冰川山谷。
  -> 尺度差异、层叠纵深、不存在力量或暴露处境；activation_evidence []。宁静的安全型
  敬畏。**V 7.4，A 5.0**，证据强。
- **威胁型**：一道高耸的风暴墙正向一座小型海滨城镇推进，并且可以看到闪电。->
  垂直方向上的巨大体量、运动中的不可控制力量、具有包围感的接近程度；activation_evidence
  ["正在推进的风暴锋面", "闪电"]。**V 3.8，A 6.8**，证据强。

## 输出约定

在共享 JSON 中报告两个尺度线索、广阔感／自我渺小感、敬畏分支、
安全性／暴露程度修正因素，以及顶层 `activation_evidence`（如果不存在则为空列表）。
顺应（accommodation）只能通过推断获得，不能被直接观察。
