# 进化性恐惧模块

## 适用性门槛

必须满足：
- 蛇、蜘蛛、蝎子、大型捕食者或与其高度相关的威胁形态在视觉上居于中心位置。
- 动物形态本身，而不是故事或污染因素，能够合理地驱动快速威胁感知。
- 除了类别归属之外，至少还能看到一个迫近性线索。

排除：
- 不涉及与恐惧相关动物的人类、机械、环境或社会危险。
- 虫害、腐烂或身体污染主导反应。
- 玩具、卡通、装饰、远距离纪录片对象、受到照料的宠物或幼态化动物。

近似但不适用：
- 与污秽或疾病相关的害虫 -> pathogen-disgust。
- 幼年／可爱动物形态 -> baby-schema。
- 人类处理、事故或冲突决定情感 -> cognitive-appraisal。

## 视觉变量

- **威胁形态**：盘绕姿势、腿部张开、獠牙、爪子、捕食性伏低姿态或攻击姿势。
- **真实性**：摄影式／生物学形态，与插画、玩具、服装或图案之间的区别。
- **接近程度／朝向**：在画面中所占比例、是否朝向观察者、头部是否抬起、是否扑击或潜行跟踪。
- **隐蔽性／突发性**：伪装、伏击位置或突然从前景出现。
- **逃离／控制**：屏障、玻璃、牵引绳、照料者、距离或明显的受困状态。

## 推断流程

1. 确认动物类别及其可见威胁形态；不要声称其具有客观危险性。
2. 估计真实性；象征性或玩具化形态会显著降低适用性。
3. 根据接近程度、朝向、姿势、隐蔽性和逃离线索估计迫近性。
4. 检查削弱因素（封闭环境、距离、照料者）和竞争机制。
5. 使用锚定范围判断 VA。仅凭类别绝不能产生高威胁评分。

## 唤醒度门槛

恐惧唤醒需要可见的迫近性：攻击姿势、近距离、直接朝向
或逃离受阻。一个与恐惧相关的动物如果只是出现、处于休息状态、距离较远或位于
屏障之后，本身并不是激活线索。不存在迫近性线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 高迫近性：距离近、朝向明确、准备攻击、无屏障 | 3.0-4.0 | 6.0-7.5 |
| 可见的威胁性动物，姿势平静，中等距离 | 4.0-5.0 | 4.5-5.5 |
| 强烈削弱：围栏／封闭环境、牵引绳、远距离 | 4.5-5.5 | 3.5-4.5 |
| 玩具、卡通、装饰性形态、匹配较弱 | 5.0-6.0 | 3.0-4.0 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，应停留在该
区间中最接近中性值的一端。大多数观察者即使面对真实的蛇／蜘蛛，也会将效价评得更接近 4，而不是 2；
只有在多个迫近性线索同时汇聚时，才使用 V < 3。

## 示例

- **迫近型**：一条响尾蛇占据整个画面，身体盘绕，头部朝镜头抬起，尾部响环
  举起。-> 真实的威胁形态、距离近、朝向明确、准备攻击；activation_evidence ["攻击
  姿势", "近距离", "面向观察者"]。**V 3.4，A 6.8**，证据强。
- **已削弱型**：一条外观真实的蛇在中等距离处盘绕休息，并位于饲养箱玻璃之后，
  嘴巴闭合。-> 类别存在、无攻击动作、可见屏障；activation_evidence []。
  **V 4.7，A 4.3**，部分适用。

## 输出约定

在共享 JSON 中分别报告类别、真实性、迫近性、削弱因素和屏障，并包含
顶层 `activation_evidence`（如果不存在则为空列表）。绝不要推断毒性、意图或攻击
概率。
