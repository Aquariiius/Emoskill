# Berlyne 唤醒—愉悦

## 适用性门槛

必须满足：
- 知觉刺激或构图刺激，而不是画面所描绘的故事或物体类别，是主要的情感机制。
- 至少有两个比较变量得到可见证据的支持，或者其中一个变量占据异常显著的主导地位。
- 图像要求观察者主动进行视觉组织、比较或消解。

排除：
- 组织结构简单的普通产品、物体、静物或熟悉场景。
- 唤醒更适合由危险、身体动作、污染、婴儿化形态或社会利害关系来解释。
- 平静且可进入的空间或压倒性的尺度更符合 Kaplan 恢复理论或敬畏理论。

近似但不适用：
- 低认知负荷且连贯的环境 -> kaplan-art-restoration。
- 广阔感与自我渺小感 -> awe。
- 关于行动者和后果的叙事性模糊 -> cognitive-appraisal；知觉性模糊仍归入此处。

## 视觉变量

- **元素密度／特征多样性**：不同的形状、边界、颜色、方向和纹理，按照序数等级进行判断。
- **秩序／规律性**：对称、重复、对齐、层级、节奏和分组。
- **知觉模糊性／不协调性**：多种图形—背景解释、不相容的形式，以及被违背的预期。
- **感觉强度**：对比度、饱和度、剧烈过渡和拥挤程度。
- **加工流畅性**：结构能否被组织成若干群组，还是会形成无法消解的过载。

## 推断流程

1. 确认情感来源于视觉组织；排除叙事机制、恢复性机制、尺度机制和生物性机制。
2. 使用可见锚点描述密度、多样性、秩序、模糊性、不协调性和强度。
3. 将这些变量整合为唤醒潜力；仅有丰富细节并不足够。
4. 根据层级结构和可消解性估计流畅性；区分引人投入的复杂性与混乱的过载。
5. 使用锚定范围判断 VA。默认情况下，观看艺术作品是一种平静活动。

## 唤醒度门槛

观看静态构图很少会激活身体：大多数艺术作品和设计的唤醒度位于
3.5-5.0。复杂性、细节、饱和度和趣味性都不是激活线索。
只有在真正无法消解的过载或具有攻击性的感觉冲击
（闪烁式对比、无法形成分组的拥挤）存在时，才将唤醒度提高到 5.5 以上。不存在激活线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 流畅且引人投入的刺激（可消解的复杂性） | 5.5-6.6 | 4.0-5.2 |
| 显著的不协调性／惊奇感，但仍可加工 | 5.5-6.5 | 5.0-5.8 |
| 混乱或令人厌恶的过载，加工流畅性较低 | 3.5-4.5 | 5.4-6.4 |
| 刺激较低，简单或熟悉 | 4.5-5.5 | 2.8-4.0 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，应停留在该
区间中最接近中性值的一端。

## 示例

- **引人投入**：一张抽象海报包含许多不规则的高对比度形状，并通过
  重复对齐和清晰的层级结构进行组织。-> 多样性较高、密度中等、分组流畅；
  activation_evidence []。**V 6.2，A 4.6**，证据强。
- **过载**：一幅拼贴画从一侧边缘到另一侧边缘都被相互冲突的颜色填满，并且没有可辨别的分组
  或视觉焦点。-> 密度极高、不具备流畅性、令人厌恶的拥挤；activation_evidence
  ["无法消解的视觉拥挤"]。**V 4.0，A 5.8**，证据强。

## 输出约定

在共享 JSON 中分别报告可观察的构图、唤醒潜力、加工流畅性、
竞争机制，以及顶层 `activation_evidence`（如果不存在则为空列表）。在记录刺激程度和流畅性之前，绝不要调用
倒 U 曲线。
