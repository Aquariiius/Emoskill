# 情绪身体语言

## 适用性门槛

必须满足：
- 人的全身或可见上半身承载主要情绪信号。
- 面部信息缺失、被遮挡、过小、被面具遮住，或仅起次要作用。
- 姿势支持一种可识别的动作倾向：接近、后退、僵住、攻击、瘫倒、庆祝、宣示或服从。

排除：
- 可辨认的面孔或面部表情占主导 -> facial-expression-affect。
- 武器、危险物体或后果决定情感 -> cognitive-appraisal。
- 画面中只出现孤立的手、含义模糊的静止姿势或很小的身体片段。

近似但不适用：
- 可见的行动主体、利害关系、冲突或物体使用 -> cognitive-appraisal。
- 以中性面孔为中心的肖像 -> no specialized skill。
- 捕食性动物姿势 -> evolved-fear-module。

## 视觉变量

- **动作情境（首先判断）**：运动、舞蹈、娱乐、庆祝、劳动、抗议、暴力或不明确——它决定效价分支。
- **扩张／收缩**：躯干是开放还是闭合；四肢是抬起，还是收拢或用于遮挡防护。
- **紧张程度／能量**：关节僵硬、肌肉蓄力、爆发性动作、松弛或瘫倒。
- **朝向**：身体前倾、后缩、转身避开、向上伸展或稳定扎根的站姿。
- **受害者／威胁检查**：只要有人正在受伤、逃跑或受到支配，就将分支转为负面。

## 推断流程

1. 确认身体可见性；应用移除测试（如果遮住面部，该信号是否仍然存在？）。
2. 在解读情绪之前，先对动作情境进行分类；运动／舞蹈／玩耍场景属于积极动作，除非可以看到受害者、武器或碰撞伤害。
3. 描述身体构型（扩张、紧张程度、朝向），不要直接命名情绪。
4. 推断主要动作倾向和一个竞争性动作倾向；静态画面可能会模拟某种情绪。
5. 使用锚定范围判断 VA；唤醒度取决于姿势能量，而不是情绪标签。

## 唤醒度门槛

唤醒度取决于可见能量：爆发性动作、僵硬蓄力、腾空阶段或全速冲刺
会提高唤醒度；站立、行走、摆姿势或松弛姿势不会提高唤醒度。缺乏动作的戏剧性氛围
并不是激活线索。不存在激活线索 -> 唤醒度 <= 4.5。

## VA 锚定范围（1-9）

| 情况 | 效价 | 唤醒度 |
|---|---:|---:|
| 庆祝、运动、舞蹈、充满活力的玩耍（无受害者） | 6.5-7.6 | 5.4-6.6 |
| 针对他人的攻击性或攻击姿势 | 2.5-3.5 | 6.0-7.2 |
| 恐惧：后缩、防护、逃跑 | 3.0-4.0 | 5.5-6.5 |
| 悲伤：瘫倒、垂头缩身式退缩 | 2.8-3.8 | 3.0-4.4 |
| 平静的中性活动或含义模糊的姿势 | 5.0-6.0 | 3.4-4.5 |

这些锚定范围是典型区间，而不是强制输出值；应在区间内进行插值。当证据不足时，应停留在该
区间中最接近中性值的一端。

## 示例

- **积极动作**：一名跑步者越过终点线，双臂高高举起，背部后弓，仍处于迈步过程中。
  -> 运动情境、身体扩张、爆发性动作、无受害者；activation_evidence ["冲刺
  动作", "双臂猛然向上举起"]。**V 7.2，A 6.2**，证据强。
- **悲伤**：一个孤单的人垂头坐在路缘石上，头部低垂，双肩塌下，静止不动。->
  身体收缩、松弛、退缩、无动作；activation_evidence []。**V 3.2，A 3.6**，
  证据强。

## 输出约定

在共享 JSON 中报告动作情境、解释前的身体构型、竞争性
倾向，以及顶层 `activation_evidence`（如果不存在则为空列表）。描述感知到的
外在表达，而不是实际的内在状态。
